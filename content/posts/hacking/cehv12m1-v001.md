## Prerequisites
- OS
- File system
- Computer network
- Data leakage
- Risk management
- TCP/IP protocols
- Information security controls
- Basic networking troubleshooting
- Data backup

## Classification of Attacks
### Passive Attacks
PA do not tamper with the data and involve intercepting and monitoring network traffic and data flow on the target network.

Examples of PA:
- Footprinting
- Sniffing and eavesdropping
- Network traffic analysis
- Decryption of weakly encrypted traffic

### Active Attacks
AA tamper with the data in transit or disrupt the communication/services between the systems to bypass or break into secured systems.

Examples of AA:
- DoS attack
- Bypassing protection mechanisms
- Malware attacks (such as viruses, worms, ransomeware)
- Modification of information
- Spoofing attacks
- Replay attacks
- Password-based attacks
- Session hijacking
- MitM attack
- DNS and ARP poisoning
- Compromised-key attack
- Firewall and IDS attack
- Profiling
- Arbitrary code execution
- Privilege escalation
- Backdoor access
- Cryptography attacks
- SQL injection
- XSS attacks
- Directory traversal attacks
- Exploitation of application and OS software  

### Close-in Attacks
CA are performed when the attacker is close physical proximity with the target system or network in order to gather, modify, or disrupt access to information.

Example of CA:
- Social engineering (eavesdropping, shoulder surfing, dumpster diving, and other methods)

### Insider Attacks
IA involve using privileged access to violate rules or intentionally cause a threat to the organization's information systems.

Examples of IA:
- Eavesdropping and wiretapping
- Social engineering
- Data theft and spoliation
- Pod slurping
- Theft of physical devices
- Planting keyloggers, backdoors, or malware

### Distribution Attacks
DA occur when attackers tampir with hardware or software prior to installation.

Examples of DA:
-  Modification of software or hardware during production
-  Modification of software or hardware during distribution
