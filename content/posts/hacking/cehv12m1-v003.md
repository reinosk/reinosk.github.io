## CEH Hacking Methodology (CHM)
This methodology helps security professionals and ethical hackers understand the varius phases followed by real hackers in order to achieve their objectives.

### Footprinting
Attacker gathers as much information as possible about the target prior to launching an attack.

### Scanning
In this phase, the attacker uses the details gathered during reconnaissance to scan the network for specific information.

### Enumeration
It is a method of intrusive probing through which attackers gather information such as network user lists, routing tables, security flaws, shared users, groups, applications, and banners.

### Vulnerability Analysis
Attackers perform VA to identify security loopholes in the target organization's network, communication infrastructure, and end systems.

### System Hacking
They first obtain information during the footprinting, scanning, enumeration, and VA phases, which they use to exploit the target system.

- Gaining access
- Escalating privileges
- Maintaining access
- Clearing logs
